Why CURD?

Unlike tools that guess where you want to go from usage patterns, CURD gives you explicit control: you save the directories you care about and reach them by keyword.

⚡

Lightning fast

No database, no ranking heuristics. A single binary reads a small YAML file and prints a path. Navigation is instant.

🎯

Predictable

A keyword always takes you to the exact same place. No surprises, no "did you mean" — you chose the bookmark, you get it.

🔖

Keyword bookmarks

Save the current directory or any path under a memorable keyword with curd save. A special default keyword needs no name at all.

⌨

Tab completion

The curr wrapper completes your saved keywords in Bash, Zsh, Fish, PowerShell, and csh/tcsh. Generate command completion with curd completions.

🌍

Cross-platform

Works on Linux, macOS, and Windows across Bash, Zsh, Fish, PowerShell, csh/tcsh, and the Windows Command Prompt.

🔒

Secure & self-contained

Config files use user-only 0600 permissions and reject .. path traversal. No external dependencies — just download and run.

Quick Start

From install to instant navigation in three steps.

1. Install CURD

go install github.com/dmcbane/curd/v2@v2.2.0   # or grab a binary from the releases page

2. Add the curr wrapper to your shell (Bash, ~/.bashrc)

function curr() {
  D=$(curd "$@")
  cd "${D}"
}

# Tab completion of saved keywords (bash only)
if [ -n "$BASH_VERSION" ]; then
  _curr_complete() {
    COMPREPLY=()
    if [ "$COMP_CWORD" -eq 1 ]; then
      COMPREPLY=($(compgen -W "$(curd ls -k)" -- "${COMP_WORDS[COMP_CWORD]}"))
    fi
  }
  complete -F _curr_complete curr
fi

Using Zsh, Fish, PowerShell, or csh/tcsh? See the installation guide for the equivalent setup with tab completion.

3. Save directories and jump between them

curd save project        # bookmark the current directory as "project"
curr project             # jump to it from anywhere
curd ls                  # list everything you've saved
curd clean               # drop bookmarks whose paths no longer exist

🐸 New in v2.2.0 — keyword tab completion for curr

The curr integration scripts now include tab completion of your saved keywords in every supported shell, with ready-to-use curr.zsh and curr.csh scripts alongside the existing Bash, Fish, and PowerShell versions.

A day with CURD

Set your bookmarks once in the morning, then bounce between projects all day.

# Morning setup
cd ~/work/backend  && curd save be
cd ~/work/frontend && curd save fe
cd ~/work/database && curd save db
cd ~/.config       && curd save config

# Throughout the day — tab-complete the keyword
curr be       # jump to backend
curr fe       # jump to frontend
curr config   # jump to config files

# Housekeeping
curd list    # see all saved paths
curd clean   # remove non-existent paths
